# Синхронизация остатков и цен на маркетплейсах

Данный инструментарий автоматически обновляет остатки и цены на товары на площадках __Ozon__ и __Яндекс.Маркет__, используя данные с сайта https://timeworld.ru

## Как работает (кратко)?

- Скачивается файл с актуальными остатками товара с сайта ```timeworld.ru```
- Сравниваются эти остатки с товарами, которые у вас есть на __Ozon__ и __Яндекс.Маркет__.
- Обновляется товар в наличии (остатки) и цена, по которой продается товар у вас.
- Работа по обновлению производится автоматически при запуске скриптов, не требует ручного ввода.

_Данные выгружаются из файла остатков с сайта ```timeworld.ru```, где хранится актуальная информация о наличии и ценах. Цены автоматически меняются на основании данных из файла._

_Артикулы (SKU) — используются внутренние коды товаров из файла остатков._



## Как запустить программу?

#### 1. Обновление данных на OZON (```seller.py```)

Откройте командную строку и выполните:

```
python seller.py
```

Скрипт:

- Скачает актуальные остатки и цены с сайта ```timeworld.ru.```
- Обновит остатки на Ozon
- Обновит цены на Ozon


#### 2. Обновление данных на ЯНДЕКС.МАРКЕТ. (```market.py```)

Откройте командную строку и выполните:

```
python market.py
```

Скрипт:

- Обновит остатки и цены для FBS

  _Продавец самостоятельно хранит товары на своём складе, товар при покупке продавец везет самостоятельно пункт приема, а доставку заказов осуществляет маркетплейс._

- Обновит остатки и цены для DBS (доставка от продавца)

  _«Доставка силами продавца» - маркетплейс выступает только как витрина и площадка для оплаты, а всё остальное (хранение, упаковка и сама доставка до двери клиента) лежит непосредственно на продавце._

```
  Для FBS и DBS используются свои идентификаторы кампании и склада.
  Обновление производится раздельно.
```
